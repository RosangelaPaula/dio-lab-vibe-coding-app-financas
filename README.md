## PRD Refinado no Copilot WEB

### 1. Contexto
O aplicativo será uma solução de organização financeira pessoal baseada em conversas em linguagem natural.  
O objetivo é simplificar o controle de gastos e metas financeiras sem depender de formulários complexos ou planilhas.  

### 2. Problema
- Usuários desistem de controlar suas finanças porque os apps atuais exigem muita entrada manual.  
- Falta de personalização e de uma experiência fluida.  
- Necessidade de uma abordagem mais natural e acessível, especialmente para iniciantes e idosos.  

### 3. Público-Alvo
- Pessoas que desejam começar a organizar suas finanças de forma prática.  
- Usuários com pouca familiaridade com tecnologia (ex.: idosos).  
- Pessoas que buscam simplicidade e acessibilidade.  

### 4. Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural.  
2. Classificação automática das transações (alimentação, transporte, lazer etc.).  
3. Definição e acompanhamento de metas financeiras.  
4. Agente Financeiro: dicas automáticas de economia personalizadas.  
5. Relatórios simples e personalizados (gráficos e resumos).  
6. Câmbio diário de moedas (euro e dólar).  
7. Suporte multilíngue: português, inglês e espanhol.  

### 5. Entregável da IA (MVP)
- **Principais telas**:  
  - Tela de chat (interação principal).  
  - Tela de relatórios (gráficos simples).  
  - Tela de metas financeiras.  
  - Tela de configurações (idioma, preferências).  

- **Recursos necessários**:  
  - Motor de NLP (Processamento de Linguagem Natural).  
  - API de câmbio para atualização diária.  
  - Sistema de categorização automática de gastos.  
  - Módulo de relatórios visuais.  

- **Validação inicial**:  
  - Testar com grupo piloto de usuários iniciantes e idosos.  
  - Avaliar clareza da linguagem e facilidade de uso.  
  - Coletar feedback sobre personalização das dicas.

## Interações com o Lovable
- Criação do APP de Finanças Pessoais com base no PRD.  
- Ajustes solicitados:  
  1. Manter o câmbio estático, mas com atualização automática.  
  2. Quando solicitado ao Chat para "guardar na Meta (indicada)", retirar o valor do Saldo, transferir para a Meta e descrever no Relatório de Finanças como Meta.  
  3. Tradução dos subtítulos no idioma solicitado.  
  4. Conversão automática do valor da moeda solicitada.  
- Execução de nova verificação de segurança para identificar vulnerabilidades.

## Resumo do App
O aplicativo de Finanças Pessoais permite:  
- Registrar gastos via chat em linguagem natural.  
- Classificar automaticamente transações por categoria.  
- Definir e acompanhar metas financeiras.  
- Receber dicas personalizadas de economia.  
- Visualizar relatórios simples e gráficos intuitivos.  
- Consultar câmbio diário de euro e dólar.  
- Usar em português, inglês e espanhol.

## Reflexão

### O que funcionou bem?
A interação com o Copilot foi muito gratificante e produtiva, mesmo sem conhecimento prévio em desenvolvimento.  

### O que não funcionou como o esperado?
A interação com o Lovable deixou a desejar em alguns aspectos. Minha falta de conhecimento em desenvolvimento contribuiu para isso.  
Além disso, tive problemas técnicos e precisei enviar o print do site de finanças pelo smartphone.  

### O que aprendi sobre conversar com IAs?
- Que conhecimento é essencial: quanto mais se sabe, mais fácil fica o entendimento com a IA.  
- Que a língua inglesa é fundamental para dar agilidade aos processos.
